# ⚙️ Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/sentiment-analysis.git
cd sentiment-analysis
```

## 2. Install Dependencies

```bash
pip install -r requirements.txt
```

## 3. Add the Dataset

Place `1429_1.csv` in the root directory of the project before running the notebook.

## 4. Run the Notebook

Open `sentiment.ipynb` in Jupyter and run all cells.
